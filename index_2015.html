Units of Measure

  • http://www.w3schools.com/cssref/css_units.asp
  • %, px, cm, mm, in, pt, pc
  • em -- relative to the font-size of the element (2em means 2 times the size of the current font)
  • ex -- relative to the x-height of the current font
  • ch -- relative to width of the "0" (zero)
  • rem -- relative to font-size of the root element
  • vh, vw -- represents 1% of the height/width of the viewport
  • vmin, vmax -- represents to 1% of viewport's smaller/larger dimension

Animation

@keyframes changeBackgroundAndText
{
    0%  25%  40%  55%  70%
    85%
    {
      top: 0px;  left: 0px;  background-color: #00ff00;
    }
    100%
    {
      top: 0px;  left: 400px;  background-color: #0000ff;
    }
}

.animation div { /* default size, shape, location*/ }

.animation #changingBoxShow:checked ~ div#changingBox
{
    animation-name: changeBackgroundAndText;
    animation-duration: 8s;
    animation-fill-mode: forwards;
}

#changingBoxShow:checked 
	~ div#changingBox > label#changingBoxShowLabel 
{ /* format text */ }
